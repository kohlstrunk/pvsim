PVSIM

PV-Modul-Simulator
- getestet an Hoymiles HMS-800W-2T
- erlaubt DC-seitige Einstellung der Einspeiseleistung

Enthält:
- WinAVR-Projekt
- Eagle-Schaltplan und Layout
- Fertigungsdaten

Entwicklungsstand
- Erprobung 2-Kanalig an beiden separaten MPPT-Kanälen
- Erprobung 1-kanalig mit parallel geschalteten MPPT-Kanälen
- Erprobung mit Si-H-Brücke 50V 50A bei 62kHz Schaltfrequenz

Perspektive:
- Umstellung auf 250kHz GaN-Halbbrücke 20A mit parallel geschalteten MPPT-Kanälen
